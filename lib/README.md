# lib-common
This repository is home to all functionality that is used by multiple components (i.e. which is not specific to a single component).


# libraries

## licensing

Everything related to a service, feature or product being usable by customer.

## logging

Everything related to logging messages.

## net

Networking, clients and servers.

## option

Configuration loaders for shared classes

## protocol

Basic implementation of UH protocol used for communication between components and transfer of files.

## third-party

Collects submodules to third-party library repositories.

## util

Utility classes

## test

Classes, functions and data used in test cases.
